# Privacy Policy

**scorm-skills** (the `scorm` skill / `scorm-skills` plugin) is a set of **static
instruction files** (Markdown) that teach AI coding assistants how to use the
[`@studiolxd/scorm`](https://www.npmjs.com/package/@studiolxd/scorm) library.

## What we collect

**Nothing.** This plugin:

- Does **not** collect, transmit, store, or process any personal data.
- Does **not** execute code, run background processes, or make network requests.
- Contains **no** telemetry, analytics, or tracking of any kind.
- Has **no** server or backend — it is plain text distributed via GitHub.

All it does is provide guidance text that your AI coding tool reads locally.

## Third parties

This plugin shares no data with any third party, because it processes no data.

## Changes

Any updates to this policy will be published in this file in the public repository:
https://github.com/studiolxd/scorm-skills
